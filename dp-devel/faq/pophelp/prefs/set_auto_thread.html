Watching Project Threads

If you wish, you can be signed up for notification on the forum threads for your projects automatically. When a post is made to such a forum thread, you will receive an e-mail notification.

Changing this setting will only apply to forum threads created in the future. If you wish to change the notification on already existing forum threads, you will need to do so manually.

Yes will sign you up for notification on each of your projects' forum threads created from now on.

No will not sign you up for notification on any of your projects' forum threads created from now on.
